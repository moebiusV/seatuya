Erlang NIF bindings for libseatuya. Requires a C compiler, Erlang/OTP, and `libseatuya` installed. Build the NIF with `cc -fPIC -shared -I$ERLANG_ROOT/usr/include -o seatuya_nif.so seatuya_nif.c -lseatuya`, then place `seatuya_nif.so` in the application priv dir (or set `SEATUYA_LIB` to the full path). Usage: `seatuya:create("dev_id", "ip", "local_key", "3.3")` returns `{ok, Ref}` on success.
