Elixir bindings for libseatuya. Wraps the compiled Erlang NIF module `:seatuya` (compile `seatuya_nif.c` from the Erlang binding directory with `cc -fPIC -shared -I$ERLANG_ROOT/usr/include -o seatuya_nif.so seatuya_nif.c -lseatuya`, and ensure the resulting `.so` is on the code path or set `SEATUYA_LIB`). Usage: `Seatuya.create("dev_id", "ip", "local_key")` returns `{:ok, ref}` on success.
