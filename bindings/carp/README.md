Carp FFI bindings for libseatuya. Requires Carp (recent version with `register`/`λ` FFI, system-include) and `libseatuya` installed. Build: `carp -x -b seatuya_example example.carp && ./out/seatuya_example`. NOTE: Carp's FFI is unusual -- it compiles to C and wraps C declarations at compile time. The exact syntax for `λ` function types and `(Str.cstr)` conversions may vary by Carp version. Set `SEATUYA_LIB` at the linker level (e.g., `LD_LIBRARY_PATH`). Usage: `(Seatuya.create "dev_id" "ip" "local_key" "3.3")` returns a device pointer or nil.
