R bindings for libseatuya. Requires a C compiler and R development headers (`r-base-dev` / `R-devel`). Build the C wrapper with `R CMD SHLIB -o seatuya_r.so seatuya_r.c -lseatuya`, then source `seatuya.R` and call `seatuya.init()`. Set `SEATUYA_LIB` to the path of the compiled `.so` if not in the current directory. Usage: `tuya_create("dev_id", "ip", "local_key", "3.3")` returns a `TuyaDevice` external pointer.
